Webbens födelse

Historien om World Wide Web börjar i slutet av 1980-talet vid det europeiska forskningscentret CERN. Det var där den brittiske forskaren Tim Berners-Lee, som senare skulle kallas "webbens fader", lade grunden för det globala informationsnätverk vi idag tar för givet.

Berners-Lees vision var att skapa ett system där information kunde delas och länkas samman på ett enkelt sätt, oberoende av var i världen den fanns. För att förverkliga denna vision utvecklade han en uppsättning teknologier som tillsammans bildade grunden för webben. HTML, eller HyperText Markup Language, användes för att strukturera information på webbsidor. HTTP, HyperText Transfer Protocol, gjorde det möjligt att överföra data mellan datorer, och URL, Uniform Resource Locator, fungerade som adresser för att hitta specifik information på nätet.

Den första webbsidan, som visade information om projektet, publicerades 1991. Det var startskottet för en utveckling som skulle förändra världen. Från att vara ett verktyg för forskare spred sig webben snabbt och blev tillgänglig för allmänheten. Idag är det svårt att föreställa sig en värld utan internet och allt det innebär.

Sammanfattningsvis kan vi säga att webben föddes ur ett behov av att dela information på ett smidigare sätt. Tim Berners-Lee och hans team vid CERN skapade de verktyg som gjorde det möjligt, och resten är historia.
